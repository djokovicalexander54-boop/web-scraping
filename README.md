# این برنامه، ربات هوشمند استخراج داده از دیجی کالا و نمایش آن در ربات تلگرام است
## ویژگی های این برنامه:
* تنها با نوشتن نام محصولات مورد نظر (به تعداد دلخواه) نام، قیمت، موجودی و نظرات محصولات توسط ربات استخراج میشود
* اطلاعات استخراج شده توسط هوش مصنوعی تحلیل میشود
* دیتاهای دریافت شده به ربات تلگرام منتقل میشود
* درصورت درخواست کاربر و نتها با کلیک روی دکمه های شیشه ای ایجاد شده توسط کاربر روی نام هر محصول، اطلاعات آن در ربات تلگرام نمایش داده میشود
* در تلگرام فرستاده میشود (xlsx) در حین اجرای برنامه اطلاعات بصورت فایل اکسل نیز ذخیره میشود و در صورت درخواست کاربر، جدول اطلاعات استخراج شده بصورت فایل اکسل

## ابزار های استفاده شده برای ساخت این برنامه:
* python (زبان اصلی ساخت)
* selenium , beautifulsoup (هدایت ربات برای استخراج اطلاعات سایت)
* openai (هوشمند کردن ربات و تحلیل اطلاعات)
* pandas , openpyxl (مشاهده اطلاعات بصورت فایل اکسل)
